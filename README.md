## ‚ú® About

A React + Vite demo app for generating and previewing link metadata (Open Graph / Twitter Cards) with live customization and multi-platform preview support.

> [!NOTE]  
> This App was completely generated by **Gemini 3 Pro** and **Claude Opus 4.5**

## ‚≠ê Features

- Fetch and display link metadata from any URL
- Live preview across multiple platforms:
	- Generic Card preview
	- Twitter/X preview
	- WhatsApp preview
	- Google Search preview
- Customize metadata:
	- Title
	- Description
	- Preview image (URL or upload)
- Copy generated meta tags (Open Graph + Twitter Cards) to clipboard
- Download preview as PNG image
- Optional CORS proxy toggle for cross-origin requests
- Dark/Light theme support
